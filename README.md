# ASKLE

## To Test:

1. Create new env.<br>
`conda create -n askle python==3.10`
2. Install [Docker](https://docs.docker.com/desktop/setup/install)
3. Makre sure docker-compose is working
`docker-compose -v`
4. Install requirements
` pip install --no-cache-dir -r ./backend/requirements.txt`
5. Run Docker (Desktop)
6. Build the app
`docker-compose up --build`
7. ☕
8. Open browser, check the website at:
`http://localhost:3000/`

## To Develop:
Please create on your own branch and pull from `main` periodically to get the latest update. <br>
```git checkout -b new-branch-name```
